## VPen
该插架是基于 jquery + pen(http://sofish.github.io/pen/) 的一个富文本编辑器.

### 调用方法 
- 引入 jquery 和 pen
- 引入 v_editor.js 和 v_editor.css
- 在文档中调用 $('editor').v_editor(options)

### 说明
该编辑器需要服务器端接口的支持, 在此不再赘述. 目前该编辑器还处于测试和 debug 状态, 如没有对此进行充分了解, 请勿用于生产环境之中.

### options
- editorSelect   
- placeholder   
- reload_alert
- debug
- stay
- text_list
- image_list    
- areaList  
- blockList
